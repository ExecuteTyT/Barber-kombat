# Frontend: Экраны Mini App по ролям

## Общие принципы

- Авторизация автоматическая через Telegram initData — пользователь не вводит логин/пароль
- Роль определяется по telegram_id — каждая роль видит свой интерфейс
- Нижний tab bar для навигации (максимум 4 вкладки)
- Глубина навигации ≤ 3 уровней: сеть → филиал → мастер
- Адаптивный дизайн под мобильные (основной сценарий)
- Поддержка тёмной и светлой темы Telegram (через Telegram.WebApp.themeParams)
- Анимации смены позиций в рейтинге (CSS transitions)
- Pull-to-refresh для обновления данных
- Скелетоны при загрузке данных

## Роль: Барбер

### Tab Bar

| Иконка | Название | Экран |
|--------|---------|-------|
| ⚔️ | Комбат | Рейтинг дня (главный) |
| 📈 | Прогресс | Накопительная выручка, ПВР |
| 📅 | История | Архив результатов |

### Экран: Комбат (главный)

**Header:** Дата, название филиала, бейдж "LIVE" (если рабочий день)

**Блок рейтинга:**
- Таблица барберов, отсортированных по total_score DESC
- Строка текущего пользователя выделена цветом (primary)
- Каждая строка: позиция (🥇🥈🥉 для top-3), имя, итоговый рейтинг (крупно)
- Мини-бар под каждым барбером: 5 цветных сегментов = 5 параметров
- Анимация при смене позиций (slide up/down)
- Клик на строку → раскрытие карточки с подробностями:
  - Выручка: 13 500₽
  - ЧС: 1.45
  - Товары: 0 шт
  - Допы: 2 шт
  - Отзывы: —

**Блок призового фонда:**
- 3 карточки в ряд: 🥇 4 900₽ | 🥈 2 940₽ | 🥉 980₽
- Подпись: "Накоплено с 1 октября"

**Блок плана:**
- Прогресс-бар с процентом
- "77% — 1 850 000₽ из 2 400 000₽"
- Подпись: "Нужно ~36 700₽/смену"

**Real-time:** WebSocket обновляет рейтинг без перезагрузки страницы

### Экран: Прогресс

**Блок ПВР:**
- Вертикальная шкала с порогами (300к → 800к)
- Текущая позиция на шкале (индикатор + сумма)
- Следующий порог выделен, показано "Осталось: 21 500₽"
- Достигнутые пороги = ✅, недостигнутые = ⬜

**Блок статистики месяца:**
- Побед в Комбате: 7
- Средний рейтинг: 95.2
- Средний чек: 2 150₽
- Товаров продано: 5
- Допов оказано: 23

### Экран: История

**Календарь (текущий месяц):**
- Каждый день отмечен цветом: 🥇 золотой / 🥈 серебряный / 🥉 бронзовый / серый
- Клик на день → карточка результата: рейтинг, место, выручка, ЧС

**Архив месяцев (список):**
- Октябрь 2024: 🏆 1-е место, 14 побед, 520 000₽, призовые 12 360₽
- Сентябрь 2024: 🥈 2-е место, 9 побед, 485 000₽, призовые 7 100₽

## Роль: Шеф-барбер

### Tab Bar

| Иконка | Название | Экран |
|--------|---------|-------|
| ⚔️ | Комбат | Рейтинг дня (как у барбера) |
| 🏪 | Филиал | Панель управления филиалом |
| 📈 | Прогресс | Личные показатели (как у барбера) |
| 📅 | История | Личная история (как у барбера) |

### Экран: Филиал

**Блок "Сегодня":**
- Выручка дня: 85 000₽
- Выручка месяца: 1 850 000₽
- План: 77% (прогресс-бар)
- В смене: 3/4 мастеров

**Список барберов (Бинго):**
- Таблица: имя, накопительная выручка, индикация ПВР-порогов
- Сортировка по выручке DESC
- Клик → детальная карточка барбера

**Лента отзывов:**
- Табы: Все / Негативные / Необработанные
- Карточка: оценка (звёзды), имя клиента, комментарий, мастер, дата
- Кнопка "Обработано" → модалка с полем комментария
- Бейдж с количеством необработанных на табе

**Сверка чеков (если модуль активен):**
- Список записей за день
- Статусы: ✅ ОК / ⚠️ Расхождение
- Фильтр по проблемным

## Роль: Владелец / Управляющий сетью

### Tab Bar

| Иконка | Название | Экран |
|--------|---------|-------|
| 🏠 | Дашборд | Обзор всех филиалов |
| 📊 | Отчёты | Управленческие отчёты |
| ⚔️ | Соревнования | Комбат, ПВР по сети |
| ⚙️ | Настройки | Конфигурация системы |

### Экран: Дашборд

**Header:** Общая выручка сети за сегодня + с начала месяца

**Карточки филиалов (вертикальный список):**
Каждый филиал — карточка:
```
┌────────────────────────────────────┐
│ 8 марта                    3/4 👤  │
│ 85 000₽ сегодня                   │
│ [████████░░] 77% плана            │
│ 1 850 000₽ / 2 400 000₽          │
│                          🔴 2 отзыва│
└────────────────────────────────────┘
```

- 🔴 = есть необработанные негативные отзывы
- Клик → экран филиала (аналог шефского "Филиал")
- Внутри филиала: клик на барбера → карточка барбера

### Экран: Отчёты

**Список отчётов (вертикальные карточки):**
- Выручка по филиалам
- Day-to-day сравнение
- Клиенты (новые/повторные)
- Бинго (все барберы сети)

**Каждый отчёт:**
- Фильтры: период (дата от/до), филиал (все / конкретный)
- Данные в виде таблиц и графиков
- Pull-to-refresh

### Экран: Соревнования

**Табы:**
- **Barber Kombat:** выбор филиала → live рейтинг + зачёт побед за месяц
- **ПВР:** все барберы сети с накопительной выручкой (сводная таблица)

### Экран: Настройки

**Секции:**

1. **Barber Kombat**
   - Ползунки весов: Выручка, ЧС, Товары, Допы, Отзывы (сумма = 100%)
   - Проценты призового фонда: 1-е, 2-е, 3-е место
   - Список допов (чекбоксы): воск, камуфляж, массаж...

2. **ПВР**
   - Таблица порогов: сумма → премия
   - Кнопки: добавить порог, удалить порог
   - Чекбоксы: считать товары, считать сертификаты

3. **Планы**
   - Список филиалов, у каждого поле "План на [месяц]"
   - Числовой ввод в рублях

4. **Филиалы**
   - Список филиалов с названиями
   - Привязка к YClients company_id
   - Telegram-группа для уведомлений

5. **Сотрудники**
   - Список всех сотрудников
   - Фильтр по филиалу и роли
   - Для каждого: имя, роль (select), филиал (select), грейд, цена стрижки
   - Добавление нового сотрудника (по Telegram ID)

6. **Уведомления**
   - Таблица: тип уведомления → chat_id → вкл/выкл
   - Настройка времени отправки

## Роль: Администратор

### Tab Bar

| Иконка | Название | Экран |
|--------|---------|-------|
| 📋 | Показатели | Статистика администратора |
| ✅ | Задачи | Список задач на день |
| 📅 | История | Архив результатов |

### Экран: Показатели

- Количество записей за сегодня (руками)
- Проданных товаров: N шт
- Подтверждённых записей на завтра: N
- Заполненных ДР: N

### Экран: Задачи

Чеклист:
- [ ] Неподтверждённые записи на завтра (кол-во, кнопка "Подтвердить")
- [ ] Незаполненные ДР (список клиентов)
- [ ] Непроведённые чеки (если модуль активен)

## Навигация и переходы

### Deep Linking (из Telegram-бота)

Бот отправляет inline-кнопку "Подробнее" с параметром:
```
https://t.me/BotName/app?startapp=kombat_branch_UUID
https://t.me/BotName/app?startapp=review_UUID
https://t.me/BotName/app?startapp=report_revenue_2024-10-13
```

Mini App парсит startapp и открывает соответствующий экран.

### Переходы внутри приложения

```
Дашборд → [клик филиал] → Экран филиала → [клик барбер] → Карточка барбера
Отчёты → [клик отчёт] → Детализация отчёта
Настройки → [клик секция] → Форма настройки
```

Навигация: React Router с history stack. Кнопка "Назад" через Telegram.WebApp.BackButton.
