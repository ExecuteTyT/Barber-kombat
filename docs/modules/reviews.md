# Модуль сбора и обработки отзывов

## Назначение

Автоматизация сбора обратной связи от клиентов, маршрутизация негативных отзывов, контроль обработки.

## Сбор отзывов

### Триггер

После завершения визита в YClients (webhook: record.completed) система отправляет клиенту запрос на оценку.

### Задержка

Отправка через 30 минут после завершения визита (клиент успевает уйти).

### Канал отправки

1. **WhatsApp Business API** (приоритет) — если настроен
2. **Telegram-бот** — fallback, если у клиента есть Telegram
3. **SMS** — крайний вариант (дороже)

### Формат сообщения

```
Барбершоп [Название]

Спасибо за визит! 🙏
Мастер: [Имя барбера]

Оцените качество обслуживания:
⭐ 1  ⭐ 2  ⭐ 3  ⭐ 4  ⭐ 5

[Ссылка на форму оценки]
```

Ссылка ведёт на Mini App (публичная форма, не требует авторизации):
- Выбор оценки 1-5
- Текстовое поле для комментария (опционально)
- Кнопка "Отправить"

## Маршрутизация

### Оценки 4-5 (позитивные)
- Сохраняются в таблицу reviews
- Учитываются в рейтинге барбера (параметр "Отзывы" в Barber Kombat)
- Не генерируют уведомлений

### Оценки 1-3 (негативные → Alarum)
- Сохраняются в таблицу reviews со статусом "new"
- **Мгновенное уведомление** через Telegram-бот → управляющему и шефу филиала
- Отображаются в Alarum-ленте в Mini App

### Формат Alarum-уведомления

```
⚠️ Негативный отзыв

Филиал: 8 марта
Мастер: Петров Л.
Клиент: Мария (+7...4356)
Оценка: ⭐⭐ (2/5)
Комментарий: "Долго ждала, стрижка не понравилась"

Время: 13 октября, 16:45

[Обработать →]
```

## Обработка негатива

### Статусы

- **new** — новый, необработанный
- **in_progress** — взят в работу
- **processed** — обработан

### Процесс в Mini App

1. Шеф/управляющий видит ленту Alarum (негативные отзывы)
2. Берёт отзыв в работу (статус → in_progress)
3. Связывается с клиентом (звонок)
4. Отмечает "Обработано" с обязательным комментарием (что было сделано)

### Контроль времени

Норматив: обработка в течение 2 часов после получения.
Если отзыв не обработан > 2 часов — повторное уведомление управляющему.

```python
# Celery task: проверка каждые 30 минут
def check_unprocessed_reviews():
    overdue = get_reviews(
        status="new",
        created_at__lt=now() - timedelta(hours=2)
    )
    for review in overdue:
        send_reminder(review)
```

### Агрегация для рейтинга

Средняя оценка барбера за день = AVG(rating) по всем отзывам за этот день.
Если отзывов за день нет — параметр "Отзывы" не влияет на рейтинг (вес перераспределяется между остальными параметрами).

```python
def get_reviews_score(barber_id, date):
    reviews = get_reviews(barber_id=barber_id, date=date, rating__gte=1)
    if not reviews:
        return None  # None = параметр не учитывается
    return sum(r.rating for r in reviews) / len(reviews)
```
